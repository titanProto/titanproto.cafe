my server.

mc.titanproto.cafe is my minecraft (generally) server. it runs a modpack that my partner and i created, called betterThanHRT (can confirm x3). betterThanHRT has gone through a few iterations, V1, V2, V3, and now V4.
V4 was originally going to be on 1.21.1, but due to sleepingserverstarter not working on that version, we needed to stick to 1.20.1. V4 is now complete and working on the server, and runs even better than V3 did! it's got just over 400 mods and is really fun (imo).
betterThanHRT V4 has now concluded, you can contact me for the modpack .zip if you'd like them ^w^! the .zip file includes all the mods and configs that you'll need, just import it through prism launcher.

the journey.

this server was built using the best price/performance cpu, motherboard and ram combo that i could find second-hand at the time. later i upgraded the ram to 16gb (from the original 8 that it was). the cpu cooler was reused from my pc, i had just recently upgraded to a watercooler on the pc becuase the little 92mm cooler was not enough oomph for my ryzen 7 (~75c+ at idle). the other parts (case, psu and case fans) were all bought new for this build (not sure i trust pre-owned power supplies :p). if i can find one, i'd like to upgrade the cpu to an i7-7700, i know the motherboard can take it, and the cooler would hopefully still be enough. problem is the 7700's are fucking expensive second hand for no real reason (i could get a better performing new cpu for cheaper, but i don't really wanna buy a new motherboard).
as for the operating system, i originally used ubuntu server, it worked fine enough, but eventually i decided to try arch on it as a bit of an experiment. this was my first time trying and installing arch (i now use it in many more places), and it went pretty well all things considered. not sure if it's just from what i learnt installing arch, but i much prefer how the server operates with arch
