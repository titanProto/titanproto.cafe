jellyPi.

my jellyfin server currently is running on a raspberry pi. it is an raspberry pi 4 with 8gb of ram. eventually i plan to open up the server to friends but currently the pi cannot transcode and i don't think my internet could currently keep up with unencoded streams to multiple people. eventually it will be moved to the same server that runs my mineccraft servers, as the cpu in that will be able to utilise hardware accelerated transcoding (unlike the pi 4).
in the current set up, the pi is attached to an external 8TB spinning disc. from memory it's a 5400rpm but i can't remember that for sure. the drive is in a 3d printed enclosure, and there is a usb fan attached to the pi to blow over itself and the drive (as the drive gets pretty toasty in our aussie summers).
